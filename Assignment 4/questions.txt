
(1) The default parameters in this assignment perform a convolution on a
    4096x4096 input using a 5x5 filter and a 32x32 input tile.

    (a) What are the dimensions of the output tile?

    32 + 5 - 1 = 36. So the dimensions of the output tile are 36 x 36

        

    (b) What is the total number of ghost cells for the entire input?

    4096 + 5 - 1 = 4100. So the dimensions of the halo of the entire input matrix is 4100 x 4100
    The number of ghost cells is 4100 x 4100 - 4096 x 4096 = 32784

        

(2) Assume we used a rectangular 5x7 filter instead. The convolution is still
    performed on a 4096x4096 input using a 32x32 input tile.

    (a) What are the dimensions of the output tile?

    height = 32 + 5 - 1 = 36
    width = 32 + 7 - 1 = 38
    So the dimensions of the output tile are 36 x 38
        

    (b) What is the total number of ghost cells for the entire input?

    height = 4096 + 5 - 1 = 4100
    width = 4096 + 7 - 1 = 4102
    So the dimensions of the halo of the entire input matrix is 4100 x 4102
    The number of ghost cells is 4100 x 4102 - 4096 x 4096 = 40984
